`GET workspaces/<workspaceId>/role/<roleName>`

Body Params: N/A

Query Params: N/A

Path Params:

- `workspaceId`: the workspaceid for the role to be created i
- `roleName`: the role name

---

### Details

This endpoint is used to get roles across a workspace.
This endpoint can be accessed only by users that are a member of the workspace. Will return the role if exists and all the users with the role.
