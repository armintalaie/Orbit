- Orbit is built with the **[Nextjs](https://nextjs.org/)** framework
  - React, TypeScript
  - Essential Dev Libraries: **[Kysely](https://kysely.dev/)**
  - Tooling: ESlint, Prettier
- Database and Auth are handled by **[Supabase](https://supabase.com/)**
- Deployed and hosted on **[Railway](https://railway.app/)**

---

- Services include a frontend UI, a REST API and a Websocket Sync server
  - Sync server is not essential for the app to function but maintains live updates across active clients (e.g. different users, different tabs and views in realtime)
    - Experimental: sync server uses **[Bun](https://bun.sh/)**
