## Create Workspaces

POST workspaces/

### Body Params

| Name   | Type   | Required? | Description                                                         |
| ------ | ------ | --------- | ------------------------------------------------------------------- |
| name   | string | yes       |                                                                     |
| config | object | no        | Customization configurations - if undefined will use default config |

### Query Param

- N/A

### Responses

- 200
- 400

### Scenarios

#### Successful request

Request:

```json
{
  "name": "acme"
}
```

Response:

- Status: 200

```json
{
 "message": "Workspace acme successfully created"
}
```

#### Unauthenticated Request

Request:

```json
{
 "name": "acme"
}
```

Response:
- Status: 200

```json
{
    "message": "Workspace acme successfully created"
}
```

#### Unauthorized Request

Request:

```json
{
    "name": "acme"
}
```

Response:
- Status: 200

```json
{
    "message": "Workspace acme successfully created"
}
```


#### Bad Request - Existing Name

Request:

```json
{
    "name": "acme"

```
Response:

```json
}
```

Request:
