## Contributing to the development

To contribute to Orbit, fork this repo locally and follow the [Getting Started](../Development/Getting-Started/Setup) instructions.

Check the currently open issues in the [Issues tab](https://github.com/armintalaie/Orbit/issues) for ideas on how to contribute. When you are ready to start working on an issue, create a new branch with a descriptive name, for example:

```
git checkout -b fix-404-page
```

After you have made the necessary changes, commit your changes and push your branch to GitHub:

```
git add .
git commit -m "Fix 404 page"
git push -u origin fix-404-page
```

Finally, open a pull request on GitHub and describe the changes you have made and mention the issue you have fixed.

### Submitting issues, bugs, or feature requests

To submit new issues, use the [Issues tab](https://github.com/armintalaie/Orbit/issues) and follow the provided template. Also, make sure you apply any relevant labels.
